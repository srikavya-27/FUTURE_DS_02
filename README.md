# FUTURE_DS_02
## Objective  
To analyze customer churn behavior using the Telco Customer Churn Dataset and discover patterns in customer retention, churn drivers, and usage trends.

## Dataset  
Telco Customer Churn Dataset  
Source: https://www.kaggle.com/datasets/blastchar/telco-customer-churn  

## Tools Used  
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- VS Code  

## Analysis Performed  
- Data preprocessing and cleaning  
- Overall churn rate exploration  
- Gender-wise churn comparison  
- Impact of contract type on churn  
- Relationship between monthly charges and churn  
- Tenure-based churn analysis  

## Outputs  
Generated visualizations:  
- churn_distribution.png  
- churn_by_gender.png  
- churn_by_contract.png  
- monthlycharges_vs_churn.png  
- tenure_vs_churn.png  

## Conclusion  
The study highlights major factors affecting customer churn and provides insights that can help businesses improve retention strategies and reduce customer loss.
